## 🛠️ 技术栈

- **Python 3.12**: 主要开发语言
- **Rust + Maturin + PyO3**: 高性能计算模块
- **ClickHouse**: 时序数据库
- **Akshare**: A股数据源
- **Docker**: 容器化部署
- **Redis**: 缓存系统

## 📋 学习前提

- 基础的Python编程知识
- 了解基本的金融概念（股票、K线等）
- 具备基本的命令行操作能力
