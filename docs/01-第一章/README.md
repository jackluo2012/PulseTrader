## 📁 本章项目结构

完成本章后，您的项目结构将如下：

```
PulseTrader/
├── .venv/                    # Python虚拟环境
├── docker-compose.yml        # Docker服务配置
├── pyproject.toml           # Python项目配置
├── requirements.txt         # Python依赖
├── .gitignore              # Git忽略文件
├── README.md               # 项目说明
├── config/                 # 配置文件目录
│   └── clickhouse/        # ClickHouse配置
├── src/                    # 源代码目录
│   ├── data/              # 数据处理模块
│   ├── strategies/        # 策略模块
│   ├── backtest/          # 回测模块
│   ├── execution/         # 交易执行模块
│   └── utils/             # 工具函数
├── tests/                  # 测试代码
├── scripts/                # 脚本文件
├── notebooks/              # Jupyter笔记本
├── data/                   # 数据文件
│   ├── raw/               # 原始数据
│   ├── processed/         # 处理后数据
│   └── logs/              # 日志文件
├── rust/                   # Rust高性能模块
│   ├── src/               # Rust源代码
│   ├── python/            # Python绑定
│   └── Cargo.toml         # Rust配置
└── docs/                   # 文档目录
```
